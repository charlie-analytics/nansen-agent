# Signal Accuracy Leaderboard

This folder tracks how accurate smart money signals turned out to be.

## Scoring Method
- Each signal is recorded with entry price on signal date
- 30 days later, exit price is fetched
- WIN = price up >5% | LOSS = price down >5% | NEUTRAL = within ±5%
